# CS188
Berkeley Fall 2018 CS188 personal repo

- PJ1 : DFS, BFS, UCS, A* (Heuristics) Search
- PJ2 : Games : Reflex agent, MiniMax, alpha-beta pruning, Expectimax, Evaulation 
- PJ3 : MDP, Q-Learning, Reinforcement Learning
